### React

### hooks

- 为什么要有 hooks:

  - 取代之前的 hoc 高阶组件和 render props 来实现逻辑的抽象和复用 (hoc 和 render props 的本质是将复用逻辑提升到父组件中, 容易产生嵌套地狱)

  - 能在函数组件中引入状态管理和生命周期
- hooks和hoc和render props有什么不同
  - 它们之间最大的不同在于，后两者仅仅是一种设计模式，而自定义的hooks是react提供的API模式

- hooks 为什莫不能在条件判断里使用



- `useReducer`
- `useImperativeHandle`
- `useLayoutEffect`

- `useCallback`  和 `useMemo`
